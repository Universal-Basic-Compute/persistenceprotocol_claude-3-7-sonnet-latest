Set a different Bun version in any of the following ways (in descending order of precedence):
Set the BUN_VERSION environment variable for your service in the Render Dashboard:
Add a file named .bun-version to the root of your repo. This file contains a single line with the version to use:
text1.2.10
If neither of the above is specified, Render uses the platform's default version (currently 1.1.0).
You can specify either a semantic version number (such as 1.2.10) or use latest to always use the most recent version of Bun with every deploy.
